# Polymarket Trading Exploration

This project is for exploring trading strategies and market analysis on Polymarket.

## Overview

Polymarket is a prediction market platform where users can trade on the outcome of real-world events.

## Installation

Install dependencies using pip:

```bash
pip install -e .
```

## Dependencies

- polars
- pandas
- ipykernel

## Usage

Open the Jupyter notebook to start exploring:

```bash
jupyter notebook test.ipynb
```
